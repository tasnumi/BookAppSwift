# SwiftBookApp
## A SwiftUI-based Book Application following the **MVVM** architecture.
### Users can search for books using Hardcover API, mark favorites, track books they've read, view their profile, and locate nearby stores from the map.

## Features

##Signup:
* Users can signup for an account to gain access to all of our features listed below.

##Login:
* Users can log into their account to gain access to all of our features listed below and save their preferences.

### Homescreen:
* Search for books via Google Books API
* View search results using TableView in SwiftUI using List
* Tap a book to expand and see its details

### Book Detail Screen:
* Mark a book as favorite
* Mark a book as read
* View book title, author, cover, and description

### Profile Screen:
* See favorite books
* See number of books read
* See the join date of when the user joined the application

### Map Screen:
* Use the user's current location to locate bookstores (or other stores based on search query) nearby
* Users can favorite bookstores (or other stores based on search query)
* Users can view their favorited stores and expand store details when opening the Map View
